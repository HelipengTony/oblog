# oblog
An Auto Loading &amp; Rendering MarkDown Based Blog System

<br/>

## Accomplished
+ Home page
+ Archive/Tags page
+ Friends link page
+ Single page
  - Reading progress bar
  - Article index
  - Markdown rendering (Github css)
+ Json based comment system
+ Comments (including reply) API
+ Posts (including parameterized fetch) API
+ Tags/Cates API
+ Pages API
+ Pages system
